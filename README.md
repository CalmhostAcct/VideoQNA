# VideoQNA
VideoQNA allows users to "Make AI QNA videos like WHYS.video", via ChatGPT (GPT-4o) and MoviePy and gTTS.

## Change API Key
You must change the OpenAI API key in the ```api_key="REPLACE_WITH_YOUR_API_KEY"``` part in the code.

## My use Case
It is currently used in the "Daily Dose of Answers" YT channel.
